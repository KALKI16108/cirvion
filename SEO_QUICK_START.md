## 12. Quick Troubleshooting

### Problem: Rankings Dropped
**Possible Cause:** Algorithm update, technical issue, content quality  
**Solution:** Check Search Console for errors, run Lighthouse audit, review content quality

### Problem: Rich Results Not Showing
**Possible Cause:** Structured data errors, formatting issues  
**Solution:** Run Rich Results Test, validate JSON-LD syntax, wait 2-4 weeks for crawl

### Problem: Low CTR Despite Ranking
**Possible Cause:** Poor meta title/description, competing results better  
**Solution:** Rewrite title/description, add unique angle, improve readability

### Problem: Traffic Increase But No Conversions
**Possible Cause:** Targeting wrong keywords, poor page UX, weak CTA  
**Solution:** Review GA4 audience, improve page speed, strengthen CTA messaging
