## Red Flags & Corrections

### Issues to Monitor
- [ ] Fluctuating rankings (investigate algorithm updates)
- [ ] Traffic drops (check Core Web Vitals, crawl errors)
- [ ] Low CTR despite ranking (improve meta description/title)
- [ ] High bounce rate (improve page content/UX)
- [ ] Low conversion rate (optimize CTA placement, form fields)

### Common Issues & Fixes
| Issue | Cause | Fix |
|-------|-------|-----|
| Not ranking | Low authority/backlinks | Build citations, acquire high-quality backlinks |
| Ranking but low CTR | Poor meta title/description | A/B test titles, rewrite descriptions |
| High bounce rate | Misleading headlines | Improve content relevance, fix expectations gap |
| Low conversions | Weak CTA | Improve CTA copy, placement, design |
| Traffic drop | Ranking loss | Check Core Web Vitals, security issues, algorithm updates |
